# INIAD++
![説明](img/discription.PNG)
東洋大学情報連携学部(INIAD)のオンライン教育システム[INIAD MOOCs](https://moocs.iniad.org/)に様々な機能を追加するGoogle Chrome™ の拡張機能です。

## 機能
* 直近の講義のお知らせ
* 講義スライドのダウンロード
* トップページに表示されているコース一覧の自由な入れ替え
* 1ページごとにメモ機能(複数可)
* メモを保存したページ一覧を表示するページ
* 外部リンク一覧
* ドライブを開くボタン
* 背景色の変更
* ヘッダおよびサイドバーの追従
(上記2つはアカウント設定ページから変更できます)
* トップに戻るボタン
* 入力文字カウント
